# ShieldAI - Cybersecurity Platform

This is a modern cybersecurity platform with essential tools and configurations set up.

## 🚀 Getting Started

### Prerequisites
- Node.js (v22.19.0 or higher)
- npm (v10.9.3 or higher)

### Installation
The project dependencies are already installed. If you need to reinstall:
```bash
npm install
```

## 📝 Available Scripts

### Development Server
Start a live development server with auto-reload:
```bash
npm run dev
```
This will start the server on `http://localhost:3000` and automatically open your `index.html` file.

### Code Formatting
Format all your code files (HTML, CSS, JavaScript, JSON):
```bash
npm run format
```

### Code Linting
Check your JavaScript code for potential errors and style issues:
```bash
npm run lint
```

## 🛠️ Development Tools

### Live Server
- Automatically reloads the browser when you save files
- Serves your files locally for development
- No build process required

### Prettier
- Automatically formats your code for consistency
- Supports HTML, CSS, JavaScript, and JSON
- Configuration in `.prettierrc`

### ESLint
- Finds and fixes problems in your JavaScript code
- Enforces coding standards
- Configuration in `.eslintrc.json`

## 🎯 Quick Start
1. Run `npm run dev` to start the development server
2. Open your browser to `http://localhost:3000`
3. Start coding! The browser will automatically reload when you save changes
4. Use `npm run format` to format your code
5. Use `npm run lint` to check for JavaScript issues
